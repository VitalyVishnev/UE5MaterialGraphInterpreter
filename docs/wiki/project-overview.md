# Project Overview

Status: V1 implemented locally

The project turns copied Unreal Engine Material and Material Function graphs into concise, readable pseudo-HLSL. It is a semantic analysis aid, not a replacement for Unreal's shader compiler.

The first workflow is: paste UE clipboard text, parse the graph, and render a stable code-like explanation while preserving uncertainty and missing dependencies. `sample-capture.html` is a local browser helper that saves full, partial, and intentionally broken clipboard fixtures for that work.

The repository fixture corpus spans UE 4.22 through 5.8. In addition to the initial SceneColor, Gerstner, Bayer, noise, texture-adjustment, height-to-normal, and biplanar graphs, it now includes Substrate coverage and legacy conversion, world-aligned texturing, a large snow/dust material, static-mesh decals, Pivot Painter foliage, flow maps, and wind. Full and physically truncated variants are represented. Separate UE 5.8 palette clipboards were verified live: 51 Math classes are preserved in the [Math Expression Registry](math-expression-registry.md); 45 supplied constant/input-data classes plus ten reference-audit additions are preserved in the [Constant and Input-Data Registry](input-data-expression-registry.md); and the [Advanced Material Expression Registry](advanced-expression-registry.md) includes the original advanced batch, documented sampler variants, and a repository-backed 15-class Substrate batch. The input-data, advanced, and Substrate raw captures are preserved under [docs/raw](../raw/ue5.8-clipboard-captures/README.md); the raw Math palette clipboard is still absent.

V1 is a static Vite + TypeScript application without a graph canvas or backend. Its original implementation boundary is preserved in [Baseline Implementation Plan](../baseline-implementation-plan.md).

The browser application now accepts full and partial clipboard text, discovers outputs, renders deterministic pseudo-HLSL, infers types in both graph directions, and reports certainty directly in copied code. Its semantics registry covers every captured UE 5.8 Math, constant/input-data, parameter, Material Attributes, texture/RVT, Substrate, atmosphere, distance-field, volumetric, and switch expression. Relevant Static Switches are specialized from clipboard defaults, grouped by their shared static source, and exposed as immediate `True/False` overrides with branch and consumer summaries. Static Switch and external-function sections are independently collapsible; external-function cards show only useful function and output names rather than clipped asset paths. Dynamic pins preserve Material Attribute selections, dropdowns, MipValue, DDX, and DDY. Optional algebra simplification folds local scalar constants and removes safe neutral operations while exact graph rendering remains the default. Long content stays inside bounded scroll regions. The centered desktop layout gives full height and most width to pseudo-HLSL, keeps Clipboard compact above Diagnostics, and stacks Clipboard, code, then Diagnostics on narrow screens. A native paste button reads the current clipboard and triggers the same immediate analysis as ordinary paste. The layout uses DPI-aware logical units and fluid typography. The complete fixture corpus is covered by automated tests; the production bundle is static and suitable for GitHub Pages.

The advanced registry also recognizes all captured scalar Noise and Vector Noise modes plus the parameterless Scalar Blue Noise source, including mode-dependent `float3`/`float4` outputs.
